# What Is Rich Interactive

added from terminal

## Project Overview

This is an interactive web experience exploring the question "What Is Rich?" through a multi-screen journey that includes user input, analysis, perspective exploration, and profile generation.

## Features

- Interactive particle background with WebGL canvas
- Multi-screen navigation system
- User input analysis with keyword matching
- Dynamic slider interactions for exploring wealth dimensions
- Rotating perspectives carousel
- Personalized wealth archetype profiling
- Social media sharing functionality
- Responsive design for mobile and desktop

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- HTML2Canvas for screenshot functionality
- Pure CSS animations and transitions

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the application.

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm start` - Start production server
- `npm run lint` - Run ESLint

## File Structure

- `what-is-rich-interactive.html` - Main interactive experience
- `app/` - Next.js application directory
- `landing-page-review-email/` - Related email templates
- Standard Next.js configuration files
